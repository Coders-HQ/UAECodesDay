# UAECodesDay

## Server

- To start the server in development mode

```bash
npm run dev
# or
pnpm dev
```

- To view the endpoints [http://localhost:5000/docs](http://localhost:5000/docs)

- To send a payload to the server <http://127.0.0.1:5000>

```json
{
    "command": "console.log(\"Hello World\")"
}
```
